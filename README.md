# Graveyard Battle Landing

Одностраничный промо-лендинг для презентации игры Graveyard Battle.

## Что внутри

- `index.html` — структура страницы
- `styles.css` — оформление и адаптивность
- `script.js` — управление слайдером скриншотов
- `gallery/` — изображения для галереи

## Как опубликовать на GitHub Pages

1. Создайте новый репозиторий на GitHub.
2. Загрузите в него все файлы проекта.
3. Откройте `Settings` -> `Pages`.
4. В блоке `Build and deployment` выберите:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - папка: `/ (root)`
5. Сохраните настройки и дождитесь публикации.

После этого GitHub Pages выдаст публичную ссылку на сайт.

## Почему YouTube может не показываться локально

Если открыть `index.html` напрямую как локальный файл, некоторые браузеры или настройки приватности могут ломать встроенный YouTube-плеер и показывать `Video player configuration error`.

Что помогает:

- открыть сайт через GitHub Pages
- либо запускать локальный просмотр через простой сервер, а не через `file://`
- если встроенный плеер все равно блокируется, использовать прямую ссылку на ролик
